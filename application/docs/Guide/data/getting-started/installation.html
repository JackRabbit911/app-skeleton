{{ __('Installation') }}

1. {{ __('create_root_folder') }}:

    mkdir {{ ''|escape }}
    
    cd {{ ''|escape }}

2. Use `curl` for download zip:

    curl -L https://api.github.com/repos/JackRabbit911/app-skeleton/zipball/master > skeleton.zip
    
    unzip skeleton.zip
    
    mv JackRabbit911-app-skeleton* site.zone
    
    rm skeleton.zip

Or use `git clone` for clone repository:

    git clone https://github.com/JackRabbit911/app-skeleton site.zone
    
    rm -Rf site.zone/.git

3. Go to site.zone folder:

    cd site.zone

4. Run the install.sh:

    bash install.sh

Or do the following manually:

    mkdir -p -m=777 storage
    
    mkdir -p -m=777 storage/logs storage/sessions storage/uploads storage/cache
    
    cp .env.example .env
    
    composer require alpha-zeta/framework

5. Edit your .env file:

{{ app.file('../.env.example') }}
